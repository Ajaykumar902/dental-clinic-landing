A modern, responsive landing page designed for a dental clinic. This project focuses on clean UI/UX, fast performance, and clear presentation of services offered by the clinic.

--Features

Fully responsive design
Hero section with call-to-action
Services section with icons
About-us section
Testimonials section
Contact form
Smooth scrolling
Modern and clean UI (CSS styled naturally)

--Tech Stack

HTML5
CSS3
JavaScript (optional for animation / form)
Google Fonts / Icons

git clone https://github.com/your-repo/dental-clinic-landing.git

2. Open the project
Just open the index.html file in your browser.
